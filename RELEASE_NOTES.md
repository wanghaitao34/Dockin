# Release Notes

## v1.0.0

### English

Dockin 1.0.0 is the first public release of the app.

Highlights:

- Introduces a macOS menu bar utility for keeping Dock activation on one selected display
- Blocks accidental Dock reveals on other displays in multi-monitor setups
- Supports bottom, left, and right Dock orientations
- Uses smoother bottom-edge guarding for the common bottom Dock layout
- Adds target display selection, launch-at-login support, and appearance preferences
- Adds multilingual UI support for Chinese, English, Japanese, Korean, Spanish, and French
- Ships as a signed and notarized Developer ID build in a DMG that includes an `Applications` shortcut

### 中文

Dockin 1.0.0 是这个应用的首个公开版本。

更新内容：

- 首次发布 macOS 菜单栏工具，用于将 Dock 的触发体验固定在用户指定显示器
- 在多显示器环境下阻止其他屏幕误触 Dock
- 支持底部、左侧、右侧 Dock 布局
- 针对常见的底部 Dock 提供更顺滑的边缘拦截体验
- 提供目标显示器选择、开机自启、外观设置等能力
- 提供中文、英文、日语、韩语、西班牙语、法语多语言界面
- 以经过签名和 Apple 公证的 Developer ID 发行包交付，DMG 内含 `Applications` 快捷方式
